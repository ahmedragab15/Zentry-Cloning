# Zentry Cloning Gaming Website  

A **high-end gaming website** inspired by **Zentry Website** that wins **Awwwards-winning design**, built with **React, Tailwind CSS, and GSAP**. This project features **modern UI elements, unique scroll animations, and stunning visuals**, delivering a sleek and immersive experience for gaming enthusiasts.  

## Features  

**Modern Component Architecture** – Modular design utilizing React for distinct sections like Hero, Features, and Story  
**Unique Scroll Animations** – Custom scroll-triggered effects powered by GSAP for an engaging, dynamic experience  
**Fluid Smooth Scrolling** – Integrated Locomotive Scroll via a custom context provider for seamless navigation  
**Hover & Interactive Effects** – Dynamic parallax elements and sleek transitions using react-parallax-mouse  
**Responsive Utility Styling** – Striking layouts and futuristic aesthetics built rapidly with Tailwind CSS  
**Optimized Performance** – Blazing fast loading and efficient module bundling powered by Vite  

## Technologies Used  

- **React.js** – Component-based frontend library with TypeScript  
- **Tailwind CSS** – Utility-first styling for advanced and responsive layouts  
- **GSAP & ScrollTrigger** – Complex scroll animations and pinned container effects  
- **Locomotive Scroll** – Premium smooth scrolling experience across all sections  
- **Vite** – Fast build tool and local development server
